How to use

      
    
        
  • Add finding: Enter a title, location, severity, and notes, then click this button to register the finding in the dashboard.
    • 
        
  • View JSON: Shows the current report data as JSON in the output panel and preview area.
    • 
        
  • Download JSON: Saves the report data as a JSON file for review or import.
    • 
        
  • Download Markdown: Exports the report in Markdown format for fast documentation or sharing.
    • 
        
  • Copy report summary: Copies a short report summary to your clipboard for quick notes or email.
    • 
        
  • Clear all data: Resets all fields, clears findings, and removes saved progress from local storage.
    • 
        
  • Export report (top header): This quick action triggers the same report preview as the View JSON button.
